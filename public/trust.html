Data We Collect

✅ We Collect:
  • • Order details (amount, chain, timestamp)
  • • Transaction hashes (public blockchain data)
  • • Merchant account info (email, company)
  • • Usage analytics (anonymized)
❌ We NEVER Collect:
  • • Private keys or seed phrases
  • • Wallet passwords
  • • Credit card information
  • • Social security numbers

Data Retention

  • • Transaction data: 7 years (regulatory compliance)
  • • Order history: Indefinite (merchant access)
  • • Server logs: 90 days
  • • Account data: Until deletion requested

Payment Flow

1
Customer initiates payment
Opens payment modal on merchant's website
2
Customer connects wallet
MetaMask, Phantom, or other Web3 wallet
3
Direct blockchain transfer
USDC sent directly to merchant's wallet (StablePay never holds funds)
4
Transaction confirmed
Verified on blockchain, merchant notified via webhook
Non-Custodial Architecture
StablePay NEVER holds your funds. Payments go directly from customer to merchant wallet via blockchain. We only facilitate the connection.
