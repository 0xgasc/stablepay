Quick Start

1. Add the checkout widget

Add this to your webpage to accept USDC payments:

<script src="https://stablepay-nine.vercel.app/checkout-widget.js"></script>
<div
  class="stablepay-checkout"
  data-amount="100"
  data-currency="USDC"
  data-merchant="your_merchant_id"
  data-recipient="0x..."
></div>

Checkout API

Create Checkout Session

Create a payment session programmatically

POST /api/v1/checkout/sessions

{
  "amount": 100,
  "currency": "USDC",
  "merchantId": "merchant_123",
  "recipient": "0x...",
  "successUrl": "https://yoursite.com/success"
}

Response

{
  "id": "cs_1234567890",
  "status": "pending",
  "amount": 100,
  "currency": "USDC",
  "paymentUrl": "https://stablepay.com/pay/cs_123",
  "expiresAt": "2024-01-01T12:00:00Z"
}

Wallet API

Create Customer Wallet

Generate invisible wallets for your users

POST /api/v1/wallets

{
  "email": "user@example.com",
  "name": "John Doe",
  "chain": "base",
  "merchantId": "merchant_123"
}

Get Wallet Balance

{
  "id": "wallet_1234567890",
  "address": "0x...",
  "chain": "base",
  "email": "user@example.com",
  "balance": {
    "USDC": "100.00",
    "native": "0.01"
  }
}

Onramp API

Get Quote

Get real-time conversion rates

POST /api/v1/onramp/quote

{
  "amount": 100,
  "sourceCurrency": "USD",
  "destinationCurrency": "USDC"
}

Create Onramp

POST /api/v1/onramp/create

{
  "amount": 100,
  "destinationAddress": "0x...",
  "email": "user@example.com",
  "paymentMethod": "card"
}
